

Linux环境下eclipse安装svn插件      
```
	Linux版本:CentOS6.7
	eclipse版本:Mars
```

具体安装步骤如下：
```
	1. help–>Install New Software…
	2. 点击Add按钮
	3. Name随便填写，我填写的是svn1.8
	4. Location:http://subclipse.tigris.org/update_1.8.x 输入完毕,点击Ok
	5. 下面列出Subclipse和SVNKit, 全部选中,点击 Next
	6. 继续Next直到Finish
	7. 重启eclipse，安装完毕
```
